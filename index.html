CSS Selector Cheatsheet

CSS Selectors are the first part of every CSS rule. They tell you which part of the html the CSS rule will be applied to. Some are straight forward and use simple terms, others group or stack html elements together.

CSS Selectors

Selector Name/Type CSS Notation Description
Type Selectors p { } CSS type selectors apply the rule to match all elements of a given type or tag name.
Class Selector .header { } CSS classes are denoted with a period are reusable and can be applied to many elements.
ID Selector #caption { } CSS ID selectors are denoted with a hash, are unique, and style only a single element.
Groups of Selectors p, h1, h2 { } Groups of selectors match multiple selectors to the same CSS rule, using a comma-separated list.
Chaining Selectors .section-heading.button { } Chaining CSS selectors lets you apply a rule only to elements that match all criteria.
Descendant Selectors section ol li { } This combination of selectors has a single space between each one and applies a rule to elements that are descended from another selector.
